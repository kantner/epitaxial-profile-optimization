# Simulation code for arXiv:2512.18064 (2025)
This repository contains MATLAB code to reproduce the figures and simulation results presented in the following paper:

A. Thayil, L. Ermoneit, L. R. Schreiber, T. Koprucki and M. Kantner:
<i>Optimization of Si/SiGe Heterostructures for Large and Robust Valley Splitting in Silicon Qubits</i>, 
arXiv:2512.18064, DOI: <a href="https://doi.org/10.48550/arXiv.2512.18064">10.48550/arXiv.2512.18064</a>

Simulations were carried out using MATLAB R2023a.
